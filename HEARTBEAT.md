```bash

git push origin main

# 2. GitHub Actions 自动触发
# - 运行 ci-cd.yml 流水线

# - 拉取最新镜像
# - 停止并删除旧容器
# - 启动新容器

# - 检查 /health 端点
# - 发送飞书通知
```

**本地部署命令** (备用方案):

```bash

# 查看日志
docker-compose logs -f app

# 重启服务

# 停止服务
docker-compose down
```
