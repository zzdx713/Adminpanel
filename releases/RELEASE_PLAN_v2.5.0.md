## 5. 部署方案

### 5.1 部署方式
**方式一: Docker Compose 部署**
```bash
cd /www/wwwroot/ai-work/
docker-compose pull
docker-compose up -d
```

**方式二: 手动部署**
```bash
cd /www/wwwroot/ai-work/
git pull origin main
npm ci
npm run build
pm2 restart all
```

**方式三: CI/CD 自动部署**
- 推送代码到 main 分支
- GitHub Actions 自动触发构建和部署

### 5.2 健康检查
```bash
# 检查服务状态
pm2 list

# 查看日志
pm2 logs

# 健康检查端点
curl http://localhost:3000/api/health
```
